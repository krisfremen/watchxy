## [0.0.4](https://github.com/krisfremen/watchxy/compare/v0.0.3...v0.0.4) - 2026-05-26

### Added

- `--commands-file` (`-F`) to load watched commands from a file (one per line)
- Filterable tab picker to jump between watched commands (`Shift-T`)
- `--manual-refresh` to skip running commands on startup until the interval or **SPACE** / **r**

### Fixed

- Tab picker layout and scrolling keep the selection visible
- Case-sensitive tab filter with vim-style `\c` ignore-case modifier
